***Задание 1***
```mermaid
sequenceDiagram

    participant U as Пользователь

    participant C as Клиент

    participant S as Сервер

  

    U->>C: Ввод команды ping

    activate C

    C->>S: ICMP Echo Request

    activate S

    Note over S: Обработка запроса

    S-->>C: ICMP Echo Reply

    deactivate S

    C-->>U: Вывод результата ping

    deactivate C
  
```

***Задание 2***
```mermaid
sequenceDiagram

    participant User as Пользователь

    participant Site as Сайт

    participant Auth as Сервис аутентификации

    participant SMS as СМС-сервис

  

    User->>Site: Вводит логин и пароль

    Site->>Auth: Проверяет логин/пароль

    Auth-->>Site: Пароль верный

    Site->>SMS: Отправка кода подтверждения

    SMS-->>User: Получает СМС с кодом

    User->>Site: Вводит код из СМС

    Site->>Auth: Проверяет код

    Auth-->>Site: Код верный

    Site-->>User: Доступ разрешен

```

***Задание 3***
```mermaid
sequenceDiagram

    participant П as Покупатель

    participant К as Кассир

    participant С as Касса

  

    П->>К: Подходит с товарами

    К->>П: Приветствует

        К->>С: Сканирует товар

        С-->>К: Покажает цену

        К->>П: Кладет в зону упаковки

  

    К->>С: "ИТОГ"

    С-->>К: Сумма к оплате

    К->>П: Называет сумму

    П->>К: Оплата (карта/наличные)

    К->>С: Проводит оплату

    С-->>К: Чек оплачен

    К->>П: Отдает чек и сдачу

    К->>П: Прощается

    П->>К: Уходит
```
